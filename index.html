Animate Elements in Multiple Sections on Scroll

See the Pen Sections Animated Text on Scroll by Brian Haferkamp (@brianhaferkamp) CodePen.

Today, something useful. This small function will set an animation class on any object inside a website section when the user scrolls to that section. We've all seen this pattern where the elements animate in on scroll. It doesn't take much JavaScript for this to happen and you can set it for any element in a section or sub-section.

I feel like this is one of the more useful pieces of code I've shared. I hope that others can use it, too.
