Sticky Transparent Header

In this video I show you how to create a sticky header using the {position: sticky} CSS property. What I love about the sticky property is that it keeps the element in the flow of the document. Normally, we use a {position: fixed} to stick an element. This is great but it causes problems with the elements around it. Using {position: sticky} we no longer have to configure padding and margin for elements around the sticky element. It all just sort of works. The only downside is IE and Edge support. But this is a progressive enhancement. In those browsers the element just won't be sticky. You could use @supports to create a feature query for browsers that can utilize the sticky property and then create fixed position on the element for browsers to fall back on.
