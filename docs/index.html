Full Height Content Section Using CSS Grid

Have you ever had a short piece of content sandwiched between a header and footer that just doesn't take up enough space to force the footer all the way to the bottom of the browser window? This happens to me all the time on redirect pages after form submissions. Here's how it looks:

What we want to do is push the footer to the bottom. We could use fixed positioning:

footer {position: fixed; bottom: 0; width: 100%;}

This is the look that we are going for but doing this takes the footer out of the flow of the document and could mess things up for us later on. Here's how it looks:

We could also try: 

footer {min-height: 100vh;}

That does the trick, but it's a little brute force for my taste. It also pushes the footer past the bottom of the browser's bottom edge. This is what it looks like:

Okay, now let's get to a better solution. I've found that the best way to do this is to use CSS Grid Layout. First, we apply Grid to our body element:

body {display: grid; grid-template-rows: auto 1fr auto;}

This code alone doesn't do anything, but it gives us the framework for the magic to happen. To make the content section take up all the available space, we need to add in a minimum height for the grid we just made. This will make it stretch from the top of the viewport to the bottom of the viewport even if there is not enough content. Here's the code with the min-height property applied to the grid:

body {display: grid; grid-template-rows: auto 1fr auto; min-height: 100vh;}

Notice that we are explicitly giving values to the three rows of our grid. The first and last are auto which means the elements will take up whatever space they require. The middle value represents our content section. That should take up all of the remaining space. We do that with the 1fr value. Voila! You have a relatively positioned grid that takes up all of the viewport's height at the very least. It will also continue to allow the main content section to grow if you put a lot of content in there and it pushes everything down. The minimum height is 100vh, so it will continue to grow as content pushes the elements down the page past the bottom of the browser window.

To center everything, just add this to the main content section:

#content {place-self: center;}

We now have center-aligned (vertically and horizontally) content inside a main content section that pushes our footer to the bottom of the browser window and doesn't mess up the flow of the document to do it.
