Centering an Element with CSS

See the Pen Ways to Handle Centering in CSS by Brian Haferkamp (@brianhaferkamp) CodePen.

Centering things in front end development has often been a pain in the neck over the years. However, advances in CSS3 over the past several years make centering kind of a breeze. (Don't tell that to the designers, though). Here is a pen showing you how to do the most obvious forms of CSS centering, from the easy text-align property to the more challenging ordeal of centering an absolute positioned element. Uncomment the CSS snippets to see the .child element move in real time. Oh, browser synce, we love you.
