Sticky Sections on Scroll (CSS Only)

I'm often amazed at the things we can do now in CSS. Here is another example. We can now use position: sticky to create sticky elements on the page. When combined with the knowledge that the elements only stay sticky within the height of their containers, we can create awesome and easy effects like this one where the headers for each section stick until the user scrolls to the next section. This used to be done with JavaScript and removing the element from the flow of the HTML document. Now, the element acts both relatively and absolutely positioned. It's cool.
