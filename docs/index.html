Image Filters Using CSS

See the Pen Filtering Effects Using CSS Filters by Brian Haferkamp (@brianhaferkamp) CodePen.

Did you know you can apply image filter effects using only CSS? With CSS3 this is possible. I created this little application to show you how it works on an image. Clicking one of the labels applies a filter using a class. It's a very cool way to art direct images one at a time or apply effects to many images at one time through a class.
