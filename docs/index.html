Front End Development Challenge #1

Today I've cooked up a front end development challenge. View the design and design specs online. Code this design so that it is:

  • Mobile responsive
  • Proportionately accurate
  • Contains no accessibility errors with the WAVE test

Note: You do not have to design the bottom of the hero labeled "Design Specifics." That is just for your reference.

When you're finished with the code, share it with me on Twitter using the card in the comments section or go to @brianhaferkamp. You can also leave a link to your online code (CodePen, Github Gist, whatever) in the comments section of the XD file above so I can take a look. If you don't like any of those options, feel free to email me a link at brian@highwaywebconsulting.com with "Front End Development Challenge 1" as the subject.

To complete this challenge successfully, focus in on accessible HTML, CSS efficiency, and accuracy to the layout. Don't overthink it. This should be a layup. Show that you know the fundamentals.
