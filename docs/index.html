Make the Web Interesting

A couple of months ago I created an application called ModSite. The goal is to make it easier for web developers and designers to make the web beautiful. On the left side of the app are different modular layouts that you can literally copy and paste into your next website or app. Simply click on each thumbnail, click the code on the right, and copy it to put it into your page or app. Mix and match the different modules to create unique layouts. The code is open source and free for you to use on any site.

Check out ModSite.
