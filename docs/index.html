Video: How to Use an API to Get Data for Your Website or Web App

I've been working a lot, both personally and professionally, with APIs and data over the past year. I have also built a JAMstack app: skmmr.news. One of the first APIs I started working with is the Unsplash API. You get access to almost 1 million free to use (for any purpose) images and the API is very well documented. I highly encourage it as a starting point if you're new to APIs. In fact, I created this video as an introduction to using APIs to get data for your website or web app. Once you're finished watching, I have a YouTube playlist of several other videos focusing on developing applications with APIs.
