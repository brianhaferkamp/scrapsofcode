YouTube-Style Album Art

See the Pen Style Header by Brian Haferkamp (@brianhaferkamp) CodePen.

I really enjoy YouTube and spend several hours a day there watching videos and (mostly) listening to music while I work. I was inspired to create this pen that shows a blurred version of the album artwork behind a small album cover with the name of the band and album. This is done using an actual image as the background image. To get the blur effect I've placed a filter: blur() on the image itself. In order to have the image behave like a background image, I used object-fit: cover which makes the image responsive&mdashvery similar to the background-size property. This could also be done by using pseudo elements to overlay a blurred layer over an actual background image in the container. To top it all off, it's responsive so the relationship of the text and album art change based on viewport width.
