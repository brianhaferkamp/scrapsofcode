Banded Features Section Using CSS Grid

I was working on a design yesterday to practice my Adobe XD skills and work out an idea for a business template. I created an interesting features section that relies on some overlapping elements. You can see them here:

I haven't really thought about the challenge of coding such an element. This is a common part of designing for the Web. Designers often don't understand all the limitations of the code. Over the past year many of those previous limitations have been taken away through the combination of Flexbox and CSS Grid Layout in CSS. Many of the things that designers can do in an image editing program developers are able to do in the code.

Here is the workup of the features section in a CodePen pen:

See the Pen Features Section by Brian Haferkamp (@brianhaferkamp) CodePen.

A couple of things to notice are the use of the transform: scale() property on the image. This gives us the illusion of the overlapping without needing to fiddle with the size properties of the image. Since the scaling is very small it shouldn't distort the image.

The other thing I wanted to point out is the use of CSS Grid to control the layout of each features section. Grid makes it really easy to create an inverted section. Just change the way the elements are laid out using a particular class. It's not difficult.

What I want to do with design ideas like this is to show how far we can go now with CSS Grid. The designs that we sent back to designers to change are now easily achievable. Let's go out and make the Web beautiful!
