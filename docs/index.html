HTML-Only FAQs

Here's an accessible, easy-to-implement FAQ pattern that doesn't use JavaScript. It's one of my favorites.

See the Pen the HTML5 Details Tag for FAQs by Brian Haferkamp (@brianhaferkamp) CodePen.

The key to this FAQ is the <details> tag in HTML 5.2. There are a couple of notable parts to this. First, the FAQ group should go into the <details> element. That's essentially your wrapper. Inside that wrapper element there is a <summary> element. That is where you put the text or icon that the user clicks. In our case it is the question for the FAQ. The final element in the group is the content that is hidden and shows on click. This can be anything you can put into HTML (text, video, audio, images, etc.) so go crazy with it. Note that anything in the <details> wrapper that is after the <summary> element will be hidden until <summary> is clicked. This allows for multiple paragraphs or elments.

It's also possible to do some basic animations using CSS. I've put those in the example, too, just to show some of the things that are possible.
