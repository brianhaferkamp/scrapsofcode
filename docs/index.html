Basic Portfolio Filtering System

See the Pen Portfolio Filtering System // jQuery // UI/UX // CSS Grid by Brian Haferkamp (@brianhaferkamp) CodePen.

Today I wanted to share a bit of code I created back in 2017. It's a very basic portfolio filtering system using jQuery hide/show to add and remove classes on click. I can't imagine it would be difficult to add some basic animations, like fade in/out.

The reason I made this was to see how far down I could boil a filtering system for a portfolio. Sometimes a simple solution is all I need. Though there are a lot of plugins available, I needed something simple that just works right out of the box.

The grid is created using CSS Grid Layout and is fully responsive. Feel free to use it in any of your projects.
