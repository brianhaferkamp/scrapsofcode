Cross-Fading Images Using CSS

See the Pen CrossFade // CSS Backgrounds by Brian Haferkamp (@brianhaferkamp) CodePen.

This is a really great effect that you can use with just CSS. Apply a crossfade that places two images on top of one another. Play with the slider to give one image more or less transparency in relation to the other. To make for a smooth transition, choose two images that are the same size. Otherwise you get a weird resizing things. I'm not sure what this could ever be used for but maybe you could animate it using JavaScript? Maybe some cool possibilities there.

Learn all about cross-fade() on Mozilla Developer Network.
